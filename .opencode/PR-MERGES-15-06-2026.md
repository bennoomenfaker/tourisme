## Processus suivi

1. Aller sur chaque PR (liens ci-dessus)
2. Cliquer sur l'onglet **"Files changed"**
3. Vérifier les modifications
4. Cliquer **"Review changes"** → **"Approve"**
5. Cliquer **"Merge pull request"** → **"Confirm merge"**
6. Revenir sur ce repo et mettre à jour les sous-modules :
   ```bash
   cd frontend && git checkout main && git pull origin main
   cd ../backend && git checkout main && git pull origin main
   cd ..
   git add frontend backend
   git commit -m "chore: update submodules (PRs mergées par Maram)"
   git push origin Faker
   git checkout main && git merge Faker && git push origin main
   git checkout Faker
   ```
